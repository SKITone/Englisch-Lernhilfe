# Englisch-Lernhilfe — Medienenglisch

Kleine Lern-App zur Vorbereitung auf die mündliche „Medienenglisch"-Prüfung.
11 Kapitel (Zukunftstechnologien) mit deutscher Zusammenfassung, ausklappbarer englischer
Fassung, Details-Fakten, Vokabeln (EN→DE) und drei Übungs-Modi (Multiple-Choice, Free-Recall,
KI-Coach) sowie einer Prüfungssimulation.

**Nutzung:** `index.html` im Browser öffnen (bzw. über die veröffentlichte Seite).
Kein Server/Build nötig — reines HTML/CSS/JavaScript, Fortschritt lokal im Browser (localStorage).
